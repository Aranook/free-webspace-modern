Qu'est-ce que GitHub ?

GitHub est une plateforme de gestion de versions et de collaboration pour les développeurs. Elle permet de :

  • Héberger des projets en utilisant Git, un système de contrôle de version.
  • Collaborer avec d'autres développeurs grâce à des fonctionnalités comme les pull requests, issues, et discussions.
  • Maintenir l'historique des modifications d'un projet.

Créer un compte GitHub

Pour utiliser GitHub, vous devez d'abord créer un compte :

  1. Rendez-vous sur github.com.
  2. Cliquez sur Sign up (s'inscrire) en haut à droite.
  3. Remplissez le formulaire avec :
    • Votre adresse e-mail.
    • Un mot de passe sécurisé.
    • Un nom d'utilisateur unique.
  4. Suivez les instructions pour vérifier votre e-mail et configurer votre compte.

Configurer un e-mail public pour protéger votre e-mail personnel

GitHub permet de masquer votre adresse e-mail personnelle en configurant une adresse e-mail GitHub générée automatiquement :

  1. Connectez-vous à votre compte GitHub.
  2. Cliquez sur votre photo de profil dans le coin supérieur droit, puis sélectionnez Settings (paramètres).
  3. Dans le menu de gauche, cliquez sur Emails.
  4. Cochez l'option Keep my email addresses private (garder mon adresse e-mail privée).
  5. GitHub générera une adresse e-mail de type : username@users.noreply.github.com.
  6. Utilisez cette adresse pour vos commits ou autres interactions publiques.

Avantages d'un e-mail GitHub privé

  • Protège votre vie privée en évitant que votre adresse personnelle soit visible publiquement.
  • Réduit les risques de spam ou d'abus liés à votre e-mail personnel.
  • Permet une gestion simplifiée des contributions sur GitHub.
